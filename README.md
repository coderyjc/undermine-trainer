# undermine-trainer

## 简介

a process to gengerte save and edit save and trainer

最近挺喜欢这个游戏的，于是打算利用自己的专业知识做一个这个游戏的存档修改程序 + 辅助。

整个项目分为两部分：

- 存档生成器 - Web本地端，纯前端
- 游戏修改器 - Lua图形界面（Cheat-Engine自动生成）
